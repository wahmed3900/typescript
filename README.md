# TypeScript App

A small TypeScript Node app that demonstrates a typed function and a build/start workflow.

## Files

- `src/index.ts` — application entrypoint
- `src/app.ts` — typed logic and exported helper
- `tsconfig.json` — TypeScript compiler configuration
- `package.json` — scripts and dependencies

## Run locally

Install dependencies if needed:

```bash
npm install
```

Build and run:

```bash
npm run build
npm start
```

Run directly with TypeScript (dev mode):

```bash
npm run dev
```
